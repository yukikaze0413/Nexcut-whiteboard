🔥 单一功率模式测试

验证最大最小功率相同时的雕刻效果

❌ 问题描述

原始问题：当用户将最大功率和最小功率设置为相同值时，期望得到单一功率的雕刻效果， 但实际结果是一片白（没有雕刻），这不符合用户的预期。

用户期望 vs 实际结果

❌ 修正前（错误行为）

最大功率: 50%
最小功率: 50%

问题：所有像素功率都被计算为50%，但由于映射逻辑错误，实际输出为0%

power = minPower + (1.0 - c/255.0) * (maxPower - minPower)
power = 50 + (1.0 - c/255.0) * (50 - 50)
power = 50 + (1.0 - c/255.0) * 0
power = 50 (但逻辑处理错误导致输出0)

结果：一片白，没有雕刻

✅ 修正后（正确行为）

最大功率: 50%
最小功率: 50%

逻辑：检测到最大最小功率相同，启用单一功率模式

if (maxPower === minPower) {
  power = c < 127 ? maxPower : 0;
} else {
  // 正常范围映射
}

结果：暗于中灰度（127）的区域使用50%功率，亮于中灰度的区域不出光

🔧 技术实现

修正的核心代码

// 修正后的功率计算逻辑 let power: number; if (settings.isHalftone) { power = c < 128 ? maxPower : 0; } else { // 灰度模式功率计算 if (maxPower === minPower) { // 当最大最小功率相同时，实现单一功率效果 // 使用127作为阈值：暗于127的像素使用设定功率，亮于127的像素不出光 power = c < 127 ? maxPower : 0; } else { // 正常的功率范围映射 power = Math.round(minPower + (1.0 - c / 255.0) * (maxPower - minPower)); } }

关键改进点

  • 单一功率检测：检查 maxPower === minPower
  • 127阈值处理：使用中灰度127作为雕刻阈值
  • 暗色像素：暗于127的像素使用设定的单一功率
  • 亮色像素：亮于127的像素功率为0，不雕刻
  • 向后兼容：不影响正常功率范围的映射逻辑

📊 使用场景

单一功率模式适用场景

  • 轮廓雕刻：只需要雕刻轮廓，不需要灰度变化
  • 统一深度：希望所有雕刻区域深度一致
  • 简化控制：避免复杂的功率变化，简化雕刻过程
  • 材料特性：某些材料在单一功率下效果更好

使用注意事项

  • 确保设置的功率值适合目标材料
  • 白色区域将完全不雕刻
  • 所有非白色区域将使用相同功率
  • 建议先用小样测试效果
